# JishnuSetia Terminal Portfolio

Welcome to my **interactive terminal-style portfolio**!  

This project is a **custom-built virtual terminal** showcasing my skills, projects, and personality through a playful, hacker-inspired CLI environment. Explore, experiment, and interact to discover everything it has to offer.  

---

## Features

- **Virtual File System (VFS)**  
  Navigate directories (`cd`, `ls`) and read files (`cat`) like a real terminal.  

- **Custom Commands**  
  - `cat` – Display file content  
  - `ls` – List directory contents  
  - `cd` – Change directory  
  - `pwd` – Print working directory  
  - `open` – Open project or social links  
  - `theme` – Switch between multiple terminal themes (`retro`, `matrix`, `dracula`, `nord`, `monokai`, `gruvbox`, `tokyonight`, `solarized`)  
  - `whoami` – Display current user identity  
  - `neofetch` – Display system information with personalized flair  

- **Interactive & Fun**  
  The terminal encourages exploration, experimentation, and discovery. Try different commands to experience the full environment.  

- **Dynamic Themes**  
  Switch between multiple terminal themes to customize the look and feel of the interface.
